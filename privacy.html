隐私政策

最后更新：2026 年 6 月 5 日

引言

华飞容工作室（以下简称「我们」）高度重视您的隐私。本隐私政策说明了我们在您访问本网站（huafeirong.com）时，如何收集、使用和保护您的信息。

访问或使用本网站，即表示您同意本隐私政策中描述的做法。如果您不同意，请停止使用本网站。

我们收集的信息

2.1 自动收集的信息

当您访问本网站时，我们的服务器会自动记录以下信息：

  • 访问日志：IP 地址、访问时间、请求的页面、浏览器类型（User-Agent）、引荐来源
  • 技术数据：屏幕分辨率、语言偏好、时区信息（如通过 JavaScript 获取）

这些数据仅用于网站运维、安全防护和流量分析，不与任何个人身份信息关联。

2.2 您主动提供的信息

当您通过联系表单与我们联系时，我们会收集：

  • 您的姓名（可选）
  • 您的电子邮箱地址
  • 您的留言内容

联系表单通过 Formspree 第三方服务处理。提交表单即表示您同意 Formspree 的隐私政策。

Cookie 和类似技术

本网站使用以下类型的 Cookie：

  • 功能性 Cookie：用于记住您的主题偏好（深色/浅色模式），通过 localStorage 存储，不会传输到服务器
  • 第三方服务 Cookie：Formspree 可能设置会话 Cookie 以处理联系表单提交。详见 Formspree 隐私政策

本网站 不使用：广告追踪 Cookie、社交媒体追踪像素、跨站追踪技术。

大多数浏览器允许您管理 Cookie 设置。您可以在浏览器设置中禁用或删除 Cookie。

第三方服务

我们使用以下第三方服务：

服务 用途 隐私政策
Formspree 处理联系表单提交 查看
GitHub Pages 网站托管（可选部署方式） 查看

这些第三方服务有各自的隐私政策。我们建议您阅读相关文档了解其数据处理方式。

信息的使用目的

我们使用收集的信息用于：

  • 通信回复：通过您提供的邮箱回复联系表单咨询
  • 网站运维：监控网站健康状态、排查技术问题
  • 安全防护：检测和防止恶意访问、DDoS 攻击
  • 内容改进：分析热门页面和用户访问模式（匿名统计）

我们 不会 将您的信息用于：定向广告、用户画像、出售给第三方。

数据存储与安全

  • 联系表单数据存储在 Formspree 的安全服务器上
  • 网站托管在 GitHub Pages（如适用），遵循 GitHub 的安全标准
  • 我们采取合理的措施保护您的信息，但无法保证互联网传输的绝对安全
  • 服务器日志保留期限不超过 90 天

您的权利

根据适用的数据保护法律，您有权：

  • 访问：请求查看我们持有的您的个人数据
  • 更正：要求更正不准确的个人信息
  • 删除：要求删除您的个人数据
  • 限制处理：要求限制或反对数据处理的某些方面
  • 数据可携带：请求以结构化格式导出您的数据

要行使这些权利，请通过联系表单与我们联系。我们将在 30 天内回复。

儿童隐私

本网站不面向 13 岁以下的儿童。我们不会有意收集 13 岁以下儿童的个人信息。如果您认为您的孩子向我们提供了个人信息，请联系我们，我们将尽快删除。

国际数据传输

我们的服务托管在中国境内服务器上。如果您从境外访问本网站，您的信息可能会被传输到中国并在此处理。中国可能有不同于您所在国家的数据保护法律。

政策更新

我们可能会不时更新本隐私政策。重大变更时，我们将在网站首页显著位置发布通知。继续使用本网站即表示您接受更新后的政策。
